发表于 2017-07-14 | 分类于 Python

IO编程

Input Stream: 数据从外面（磁盘、网络）流进内存
Output Stream：数据从内存流出外面
外设CPU与内存速度不匹配的问题：

  • 同步IO：程序与数据写入按线性运行，一方必须等待另一方做完
  • 异步IO：程序（CPU执行）与数据写入按并行运行。

两者的最大区别是是否 等待IO执行的结果,
异步IO具有相当复杂的编程模型：回调模式、轮询模式等
