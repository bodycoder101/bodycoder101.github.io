语句8错误的原因，并不是所有的雇员都是经理，小不能赋值给大；
语句9正确的原因，所有的经理都是雇员，大（子类，大指的是域值大）可以赋值给小（父类）；
总结：只能大赋小
语句10正确的原因，父类赋值给子类必须进行强制类型转换，编译通过（本身staff[0]就是存储Manager类对象，只不过staff是Employee类的对象数组）

注意：在将父类转换为之类之前，应该使用instanceof进行检查。
